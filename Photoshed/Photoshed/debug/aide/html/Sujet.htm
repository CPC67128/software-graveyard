Sujet du projet :

La segmentation consiste à retrouver les parties homogènes (même type de couleur, intensité sensiblement la même) et ayant un sens dans une image. Segmenter une image est aussi la première étape vers la compréhension, par un ordinateur, d'une image.

Exemple : dans une image représentant une salle de classe on tentera de retrouver les régions correspondant aux tables, aux chaises, aux murs, aux fenêtres, ...etc.

Le but du projet tutoré consiste à développer des outils permettant d'effectuer une première analyse afin d'obtenir une segmentation acceptable de l'image.

Travail

Le travail s'effectuera en 3 étapes :

  • détection des contours des régions élémentaires (Algorithme du Watershed par piles ordonnées),
  • fermeture des contours,
  • fusion des régions en régions plus grandes suivant certains critères statistiques (moyenne, variance),

Il sera également nécessaire de développer une interface graphique permettant de visualiser les images traitées.

Le développement s'effectuera en langage Java sur PC / WindowsNT.
